# Prototipo interactivo - Flujos

## Flujo 1: Registro

1. Usuario abre modal desde "Comienza hoy" o "Registrarse".
2. Selecciona rol (`desarrollador` o `empresa`).
3. Completa formulario con validación en tiempo real.
4. Se crea sesión local y entra al dashboard correspondiente.

## Flujo 2: Login

1. Usuario abre modal → pestaña Login.
2. Ingresa email + contraseña.
3. Si credenciales son válidas, accede a dashboard.

## Flujo 3: Búsqueda avanzada

1. Visitante va a Explorar.
2. Define query y filtros (experiencia, budget, modalidad, tecnologías).
3. Visualiza resultados de desarrolladores y proyectos.
4. CTA de bloqueo: "Inicia sesión para contactar/aplicar".

## Flujo 4: Match y solicitudes

1. Empresa publica proyecto o revisa sus proyectos.
2. Abre botón "Ver matches" por proyecto.
3. Se calcula coincidencia por skills y se muestra badge de match.
4. Solicitudes se gestionan en tabs (pendientes/aceptadas/enviadas/recibidas).

## Flujo 5: Mensajería

1. Usuario abre sección Mensajes en dashboard.
2. Selecciona contacto en panel izquierdo.
3. Envía texto y enlace adjunto opcional.
4. Conversación se renderiza en panel derecho.

## Flujo 6: Configuración y pagos

1. Usuario abre Configuración.
2. Ve tarjetas de datos personales, notificaciones y seguridad.
3. Empresa dispone de bloque de métodos de pago.

## Flujo 7: Valoraciones

1. Tras colaborar, usuarios ven panel de valoraciones.
2. Consultan reviews bidireccionales con estrellas y comentario.
